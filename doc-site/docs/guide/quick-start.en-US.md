# Quick Start

## Installing dependencies

:::code-group

```bash [npm]
$ npm install @webrock/av-cliper
```

```bash [pnpm]
$ pnpm add @webrock/av-cliper
```

:::

If your project relies on multiple packages under `@webrock/*`, be sure to keep their version numbers consistent.
