# 视频剪辑

基于 WebAV 项目提供的 `@webrock/av-canvas` 包能**在浏览器环境中**快速实现**视频剪辑**功能。

该 DEMO 演示视频剪辑软件的基础能力

- 添加/删除素材（视频、音频、图片、文字）
- 分割（裁剪）素材
- 控制素材在视频中的空间属性（坐标、旋转、缩放）
- 控制素材在视频中的时间属性（偏移、时长）
- 实时预览播放
- 纯浏览器环境生成视频
